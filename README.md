# Implementation of Artificial Neural Network (ANN) and Image Classification using Convolutional Neural Network (CNN)

## Project Overview
This project demonstrates the implementation of two important Deep Learning techniques: Artificial Neural Networks (ANN) and Convolutional Neural Networks (CNN). The ANN model is used for prediction on a housing dataset, while the CNN model is used for image classification on the MNIST handwritten digit dataset.

## Objectives
- Implement an Artificial Neural Network (ANN) for prediction tasks.
- Implement a Convolutional Neural Network (CNN) for image classification.
- Understand the architecture and working of deep learning models.
- Evaluate model performance using appropriate metrics.

---

## Project 1: Artificial Neural Network (ANN)

### Dataset
- Housing.csv

### Objective
To build an ANN model for predicting values using housing data.

### Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- TensorFlow/Keras
- Matplotlib

### Steps Performed
1. Loaded and explored the housing dataset.
2. Encoded categorical features.
3. Split the dataset into training and testing sets.
4. Applied feature scaling.
5. Built and trained an ANN model.
6. Evaluated model performance.
7. Visualized training and prediction results.

### Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score
